PRACTICA 4 - Proyecto personal 

Proyecto realizado en Gephi en el que, a partir de transacciones de Bitcoin en tiempo real, se construye una red donde los nodos representan direcciones y las aristas las transacciones entre ellas.

El análisis permite identificar patrones y relaciones dentro del sistema, observándose nodos con un número muy elevado de enlaces.
Esto ocurre porque muchas de esas direcciones pertenecen a exchanges o servicios centralizados, que actúan como intermediarios y 
concentran un gran volumen de transacciones, funcionando como hubs dentro de la red.
