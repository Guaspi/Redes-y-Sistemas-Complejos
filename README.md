**PRÁCTICA 1 – Análisis de red**

En esta práctica se realiza el estudio estructural de una red mediante métricas como grado, centralidad, densidad o componentes conexas, con el objetivo de comprender su organización interna y detectar nodos relevantes.
**
PRÁCTICA 2 – NetLogo**

En esta práctica se trabaja con NetLogo para simular modelos basados en agentes, analizando el comportamiento dinámico de sistemas complejos y observando cómo surgen patrones globales a partir de interacciones locales.

**PRÁCTICA 3 – Poda y visualización de redes**

En esta práctica se aplican técnicas de poda para simplificar redes complejas, eliminando elementos poco relevantes y mejorando su interpretación visual, facilitando así el análisis y la identificación de estructuras significativas.


**PRÁCTICA 4 - Proyecto personal **

Proyecto realizado en Gephi en el que, a partir de transacciones de Bitcoin en tiempo real, se construye una red donde los nodos representan direcciones y las aristas las transacciones entre ellas.

El análisis permite identificar patrones y relaciones dentro del sistema, observándose nodos con un número muy elevado de enlaces.
Esto ocurre porque muchas de esas direcciones pertenecen a exchanges o servicios centralizados, que actúan como intermediarios y 
concentran un gran volumen de transacciones, funcionando como hubs dentro de la red.
